ionPredicción y Dimensionamiento de Sistemas Fotovoltaicos Off-Grid
Este proyecto desarrolla un modelo basado en SARIMAX para predecir la irradiación solar diaria en Argentina con un horizonte de un año. Con esas predicciones, se realiza el dimensionamiento de sistemas fotovoltaicos autónomos (off-grid), optimizados para condiciones de días nublados consecutivos.

Instalación
Para instalar las dependencias necesarias, ejecutá:
pip install -r requirements.txt
Uso
Activá el entorno virtual:
.\venv\Scripts\activate
